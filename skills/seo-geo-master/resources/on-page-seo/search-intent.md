# On-Page SEO: Search Intent

Search intent reveals *why* the user is searching. Google ranks the format that best satisfies the intent. If you target the right keyword with the wrong format, you will never rank.

## The 4 Primary Intents

### 1. Navigational
- **Goal**: Reach a specific site or page.
- **Example**: "facebook login", "hubspot pricing".
- **Format**: Brand pages, login portals.

### 2. Informational
- **Goal**: Learn something or find an answer.
- **Example**: "how to optimize images for web", "what is structured data".
- **Format**: Blog posts, guides, tutorials, knowledge bases. Required for Topical Authority.

### 3. Commercial Investigation
- **Goal**: Comparing products/services before purchasing.
- **Example**: "mailchimp vs convertkit", "best SEO tools 2026".
- **Format**: Comparison tables, listicles, review pages.

### 4. Transactional
- **Goal**: Ready to buy or sign up.
- **Example**: "buy iphone 16 pro max", "hire seo consultant".
- **Format**: Product pages, service landing pages, pricing checkouts.

## Agent Execution Strategy
Before writing or optimizing a page, Google the target keyword. If the top 10 results are all Listicles (Commercial), do not write a Landing Page (Transactional). Mirror the intent of the Top 3 results.
