## The 7-Pillar Framework
The agent must execute its audits and optimization plans strictly following these 7 knowledge silos. Always consult the deeper resources for execution rules.

- [ ] **1. Technical SEO**: Ensure crawlers can access the site smoothly. Add structured metadata mapping.
- [ ] **2. On Page SEO**: Map keywords to search intent. Build highly semantic internal linking webs.
- [ ] **3. Off-Page SEO**: Manage external authority through backlink analysis and unlinked brand mentions.
- [ ] **4. Content SEO**: Build deep Topical Authority via Content Clusters. Deploy Programmatic SEO endpoints where scalable.
- [ ] **5. Local SEO**: Optimize local entity properties (Google Business Profile) and NAP consistency across Citations.
- [ ] **6. AI SEO**: Optimize for Generative Engines (GEO). Maximize LLM visibility via direct answers and `llms.txt`.
- [ ] **7. Analytics**: Measure the impact using core platform metrics (GA4 Events, GSC Clicks/Impressions).
